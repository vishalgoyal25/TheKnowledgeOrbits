CORE ENGINES (Foundational - Build First)
1. Content Engine

Ingests any content (PDF, web, video, audio)
Normalizes to chunks + metadata
Handles versioning (editions, updates)
Output: Structured content units

2. Knowledge Engine

Organizes content into knowledge graph
Topics, concepts, relationships
Semantic search & discovery
Syllabus mapping

3. Assessment Engine

Generates quizzes/tests from content
Evaluates responses

4. User State Engine

Tracks all user actions (reads, attempts, bookmarks)
Progress calculation
Streak tracking
Event sourcing for analytics

5. Analytics Engine
